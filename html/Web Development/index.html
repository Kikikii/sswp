JAYDEN'S WEB DEVELOPMENT GUIDE

The main areas of web development:



Html

* HTML stands for hyper text markup language. HTML describes the content of a
web page such as: headings, paragraphs, images, links and lists. Web browsers
are used to translate html content into visible content. W3 schools is a wonderful
resource for learning HTML. Check out the HTML Tutorial.


JavaScript

* JavaScript is the programming language for the web. JavaScript allows
you to make your web pages interactive. W3 schools is a wonderful
resource for learing JavaScript. Check out the JS Tutorial.


Css

* CSS stands for cascading style sheets. CSS defines how html content
should be displayed such as: color, font, border, etc. W3 schools is a
wonderful resource for learning CSS. Check out the CSS Tutorial.
